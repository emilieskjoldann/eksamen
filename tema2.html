Grundlæggende web

I tema 2 blev vi introducerer til de mest anvendte redskaber i multimediedesign, vi lærte hvordan Visual Studio Code fungerer og hvordan man sætter en webside op i html og css. Vi fik vist hvordan man uploader til FileZilla, hvordan man Html-validerer og CSS-validerer.

Vi lærte om de forskellige designprincipper, design udvikling, fonte og tekstopsætning. Vi lærte at bruge XD til udvikling af mock-ups, lave wireframes og layoutdiagram.

Studiestartsprøve

I denne opgave fik jeg brugt min nye viden om HTML og CSS, vi fik alle de samme materialer og skulle derefter sætte hjemmesiden op ud fra wireframe, mock-ups og layoutdiagram.

Ved hjælp af wireframe og mock-ups kunne jeg se hvordan hjemmesiden skulle ende med at se ud, layoutdiagrammet hjalp mig med hvordan jeg skulle kode/stille det op inde i HTML og CSS

Temaopgave

I denne opgave brugte jeg min viden om hvordan man starter en designproces, jeg startede med at lave mindmap derefter, moodboard, styetile, mock-ups, wireframes og layoutdiagram. Denne proces gav mig en ny viden til hvordan man går fra idé til færdigt produkt.
